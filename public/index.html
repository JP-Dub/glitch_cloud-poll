Welcome to Cloud Poll. A user account is not required to view and vote on user polls. To visit the Poll Vault, select the link in the top left hand corner. You can view all the polls created or select a user from the top right hand corner to view the polls they created.

To create and modify your own polls, please create a user account by clicking on the link in top right hand corner. After creating an account you will be returned to this page where you can click the Account button and log in.
